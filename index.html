EasyZip Python Module

EasyZip is a lightweight Python module that simplifies working with zip files. It provides a user-friendly interface for creating zip archives, adding files and directories to them, and extracting their contents.

Installation

EasyZip is not available on PyPI, so you need to manually install it:

  1. Clone the Repository: Clone the `easyzip` GitHub repository to your local machine using Git:
    2. git clone https://github.com/yourusername/easyzip.git

    Replace `yourusername` with your GitHub username or the URL of your repository.

  2. Install the Package: Navigate to the project directory and use `pip` to install the package locally:
    3. cd easyzip pip install .

    The `.` at the end of the `pip install` command tells it to install the package from the current directory.

  3. Verify the Installation: To verify that the package is installed, you can check the installed packages with `pip list`, and you should see `easyzip` listed.

Usage

To use EasyZip in your Python code, you can import it like this:

from easyzip import EasyZip

Creating a Zip Archive

with EasyZip('example.zip') as ez:
    ez.add_file('file1.txt')
    ez.add_directory('my_directory')

Extracting Zip Content

with EasyZip('example.zip') as ez:
    ez.extract_to('extracted_directory')
